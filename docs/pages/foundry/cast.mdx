## wallet

new wallet feature for managing private keys. Interactive mode will prompt you for private key and a password for encryption.

```sh
$ cast wallet import your-account-name --interactive
```

use when deploying a contract or signing a transaction:

```sh
$ forge script <script> --rpc-url <rpc_url> --account <account_name> --sender <address> --broadcast
```
